Introduction

HTML is a hyper text markup language used to markup the text of the webpages. HTML5 is the latest version of HTML. HTML is basically a language which describes the structure and content inside webpages.

Tags and elements

Elements: An element have an opening and a closing tag unless it's not a void or empty element. empty element don't have a closing tag.

Tags: An opening tag start a element. everything that has to be written in the element should start after opening tag. A closing tag close the element, anything after closing tag won't be consider inside the element.
